## 工具功能说明

### 📝 移位

**功能：** 将输入的字符按照字母表顺序循环移位进行加解密。

**使用方法：**

1. 设置偏移量（-25 ~ 25 的整数），正数表示右移加密，负数表示左移
2. 在密文框输入文本，点击"↓ 解密"获得明文
3. 在明文框输入文本，点击"↑ 加密"获得密文

**可选功能：**

- 数字移位：勾选后 0-9 也参与循环移位

**使用示例：**

偏移量设置为 3，输入明文 `Hello`，加密后得到 `Khoor`

---

### 📝 自定义密码表

**功能：** 根据自定义的密文-明文对照表进行加解密。

**使用方法：**

1. 在密钥区域设置映射关系（密文元素 → 明文元素），使用空格分隔各元素
2. 在上方输入框输入密文，点击"↓ 解密"获得明文
3. 在下方输入框输入明文，点击"↑ 加密"获得密文

**可选功能：**

- 保留未匹配元素：开启后，遇到不在密钥表中的元素时会原样保留；关闭则报错

**使用示例：**

密钥设置为：`A B C D` → `1 2 3 4`
输入密文：`BACD`
点击解密后得到：`2134`

---

### 📝 固定密码

**功能：** 一个可扩展的固定密码工作台。进入页面先显示“工具选择卡片”，支持搜索并切换到具体工具；每张卡片都能独立工作，可并行处理多个密码任务。

**使用方法：**

1. 在卡片中搜索并选择需要的固定密码工具
2. 工具页头部带有发光背景板，显示工具名并提供“返回工具选择”
3. 鼠标移出卡片组的外侧边缘，会在对应外边出现“+”外部添加按钮
4. 左/上外边添加会前插，右/下外边添加会后插（最多 4 张）
5. 移动端提供固定“添加卡片”入口（默认后插）
6. 每张卡片可独立输入和转换，互不影响

**支持的密码类型：**

- 摩斯密码 📡
- 栅栏密码 🧱
- 培根密码 BA
- ROT13 🔄
- A1Z26 🔢
- 二进制 ASCII 💻
- 十六进制 ASCII 🔤

## 主题

网站支持浅色/深色主题切换。首页提供手动切换按钮，子页面会自动跟随该偏好。

当用户未手动设置时，系统在 19:00-07:00 自动启用深色主题；手动切换后会写入 `theme_manual_override` 并在全站同步生效。
