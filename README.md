## 工具功能说明

### 📝 一对一密码表

**功能：** 根据自定义的密文-明文对照表进行加解密。

**使用方法：**

1. 在密钥区域设置映射关系（密文元素 → 明文元素），使用空格分隔各元素
2. 在上方输入框输入密文，点击"↓ 解密"获得明文
3. 在下方输入框输入明文，点击"↑ 加密"获得密文

**可选功能：**

- 保留未匹配元素：开启后，遇到不在密钥表中的元素时会原样保留；关闭则报错

**使用示例：**

密钥设置为：`A B C D` → `1 2 3 4`
输入密文：`BACD`
点击解密后得到：`2134`
