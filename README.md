## 工具功能说明

### 📝 移位

**功能：** 将输入的字符按照字母表顺序循环移位进行加解密。

**使用方法：**

1. 设置偏移量（-25 ~ 25 的整数），正数表示右移加密，负数表示左移
2. 在密文框输入文本，点击"↓ 解密"获得明文
3. 在明文框输入文本，点击"↑ 加密"获得密文

**可选功能：**

- 数字移位：勾选后 0-9 也参与循环移位

**使用示例：**

偏移量设置为 3，输入明文 `Hello`，加密后得到 `Khoor`

---

### 📝 一对一密码表

**功能：** 根据自定义的密文-明文对照表进行加解密。

**使用方法：**

1. 在密钥区域设置映射关系（密文元素 → 明文元素），使用空格分隔各元素
2. 在上方输入框输入密文，点击"↓ 解密"获得明文
3. 在下方输入框输入明文，点击"↑ 加密"获得密文

**可选功能：**

- 保留未匹配元素：开启后，遇到不在密钥表中的元素时会原样保留；关闭则报错

**使用示例：**

密钥设置为：`A B C D` → `1 2 3 4`
输入密文：`BACD`
点击解密后得到：`2134`

## 主题

网站支持浅色/深色主题切换，默认为浅色。点击右上角的切换按钮可手动切换。

深色主题会在 19:00-07:00 自动启用（仅首次访问时生效），手动切换后会记住用户偏好。
